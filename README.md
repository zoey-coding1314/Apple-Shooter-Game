A 2D archery game developed in C++ using SplashKit.

✨ Features

• Shoot apples with a bow
• Score tracking
• Collision detection
• Multiple game states
• Sound effects
• Interactive UI

🛠 Technologies

• C++
• SplashKit
• Object-Oriented Programming
